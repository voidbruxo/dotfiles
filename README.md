## dotfiles 👨🏻‍💻🌟💻
These are my minimal dotfiles that I use daily. They consist of a simple qtile config that is designed to be minimalistic, without any unnecessary bloatware. It is not intended for ricing, but rather for daily use with low dependencies.

## Tools 🛠️

- Terminal: 🏢 alacritty, 🐈 kitty
- File Manager: 📁 ranger
- video player : mpv 🎞️
- Editor: ✍️ lazyvim , 🪄 doom emacs
- Wallpaper Set: 🖼️ nitrogen
- Run Prompt: 🚀 rofi, 🪟 qtile run
- Compositor: 🎨 picom
- Font for Bar: 💼 JetBrainsMono Nerd Font
- Screenshot Tool: 📸 scrot
- Lock Screen Tool: 🔒 slock
- backlight control : xbacklight / brightnessctl ☀️
- TL;DR :: Feel free to customize and use these dotfiles according to your needs. 😊
- my Linux Distro of choise : artix linux (dinit) 👨‍💻
